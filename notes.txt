## How to run

  npm install
  npm run dev

Then open http://localhost:5173 in your browser.

## Self-written vs. AI-assisted

The component structure, hook logic, and Tailwind class choices were written with AI assistance (Claude Code). The overall architecture, data shape, and feature scope follow the PROJECT_PLAN.md specification.

## One thing to improve with more time

Add drag-and-drop reordering so users can manually arrange tasks within the same priority tier. Currently "Plan My Day" sorts by priority but gives no control over order within High/Medium/Low groups. A library like @dnd-kit/core would handle this cleanly without heavy custom pointer logic.

## Plan My Day — implementation note

"Plan My Day" uses a pure JS sort (High → Medium → Low). No external API or AI dependency — simple, reliable, and instant. An API-based version was considered but ruled out: the value add is low for a single-day planner and would introduce network failure modes.
